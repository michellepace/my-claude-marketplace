### **Lora + Jost** — Intentional modern contrast

| | |
| :-- | :-- |
| **Relationship** | Diagonal — contrasting pair (Dynamic Contrast Serif → Geometric Linear Sans) |
| **Why it works** | Different skeleton *and* flesh — Lora is Dynamic with contrast and serifs, Jost is Geometric and linear sans. That separation on both axes creates clear hierarchy without competing. |
| **Best for** | Hero headings, brand statements, section titles |
| **Headings** | Weight `500`–`600`, tracking `-0.02em` to `-0.03em` |
| **Mood** | *Clean and intentional — modern minimalism with warmth underneath* |

### **Lora + Source Sans 3** — Quiet shared backbone

| | |
| :-- | :-- |
| **Relationship** | Same column — harmonious pair (both Dynamic) |
| **Why it works** | Same skeleton — both have open apertures and diagonal stress. The difference is flesh only: Lora adds contrast and serifs, Source Sans 3 stays uniform and bare. They feel related, not identical. |
| **Best for** | UI text, navigation, subheadings, captions — anywhere type should recede |
| **Headings** | Weight `600`, tracking `-0.01em` |
| **Watch out** | Source Sans 3 is one of the most widely used fonts on the web. Colour or weight can help establish identity. |
| **Mood** | *Understated and functional — the typography stays out of the way* |

### **Lora + Cormorant Garamond** — Too close for comfort

| | |
| :-- | :-- |
| **Relationship** | ❌ Same cell — both Dynamic · Contrast Serif |
| **Why it doesn't work** | Same skeleton *and* flesh — both have open apertures, diagonal stress, thick-thin contrast, and serifs. They're different enough to notice but not different enough to create hierarchy. |
| **Best for** | Cormorant Garamond works well as a display serif heading — just not paired with Lora. Try it with a Linear Sans instead. |
| **Mood** | *Redundant — two similar voices competing for attention* |

## The Matrix

```text
           Dynamic            Rational           Geometric
           ────────────────   ────────────────   ────────────────
Contrast   │              │   │              │   │              │
Sans       │              │   │              │   │              │
           ────────────────   ────────────────   ────────────────
Contrast   │ *LORA*       │   │              │   │              │
Serif      │ CORMORANT G  │   │              │   │              │
           ────────────────   ────────────────   ────────────────
Linear     │              │   │              │   │              │
Sans       │ SRC SANS 3   │   │              │   │ JOST         │
           ────────────────   ────────────────   ────────────────
Linear     │              │   │              │   │              │
Serif      │              │   │              │   │              │
           ────────────────   ────────────────   ────────────────

✅ Lora → Jost: diagonal = contrasting pair
✅ Lora → Source Sans 3: same column = harmonious pair
❌ Lora → Cormorant Garamond: same cell = too similar
```

## 🤔 Choose If

| Pick | If you want... |
| :-- | :-- |
| **Jost** | The clearest visual break between heading and body. Modern, minimal, intentional. |
| **Source Sans 3** | The typography to disappear completely. Maximum readability, zero personality clash. |
| **Cormorant Garamond** | ❌ Avoid — same matrix cell as Lora. Gorgeous font, wrong partner. |

## Final Recommendation

**Jost** for headings if you want clear contrast — it's the strongest visual break from Lora's serif body text. **Source Sans 3** if you need a secondary sans that blends quietly (navigation, captions, UI labels). Both are widely available and well-supported as variable fonts.

Avoid Cormorant Garamond as a Lora pairing — same matrix cell. If you love Cormorant Garamond's display qualities, consider it as the *primary* and pair it with a Linear Sans like Outfit or Source Sans 3 instead.
