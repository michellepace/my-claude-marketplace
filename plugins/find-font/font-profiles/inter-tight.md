# Synopsis: Inter Tight

Specialised version of Inter with tighter spacing, designed by Rasmus Andersson for display usage. Includes Roman and Italic styles.

## Key Characteristics

- **Classification:** Sans serif (display)
- **Character:** Tighter spacing variant of Inter, tuned for display usage
- **Intended use:** Display
- **Family:** Sibling of Inter — a specialised tighter-spaced cut of the Inter family
- **Adoption (2026-04-22):** 247M weekly serves, 163,000+ websites

## Technical

- **Variable font (1):** Weight (`wght`) 100–900
- **Styles:** Normal + Italic
